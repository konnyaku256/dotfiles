# dotfiles
This is my dotfiles repository for macOS.
You can quickly setup my development environment.

## How to setup
```
cd ~
git clone https://github.com/konnyaku256/dotfiles.git
cd dotfiles
./setup.sh
```

## Settings for iTerm2
General -> Preferences
1. Check 'Load preferences from a custom folder or URL'
2. Set path '~/dotfiles'
